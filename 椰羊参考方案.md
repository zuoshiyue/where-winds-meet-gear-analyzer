## 🚀 实现方案

### 方案 A：纯 Web 版 (当前路线)

**优势：**
- 无需安装
- GitHub Pages 托管
- 跨平台

**限制：**
- 无法自动截图
- 需要手动上传

**实现步骤：**
1. 保持当前 Web 架构
2. 添加批量上传优化
3. 添加快捷键支持 (Ctrl+V 粘贴截图)
4. 添加燕云十六声 logo

### 方案 B：Electron 桌面版 (推荐)

**优势：**
- 可以访问系统 API
- 支持游戏内截图
- 支持全局快捷键
- 可以添加游戏内覆盖层

**技术栈：**
- Electron 28+
- Vue 3 + Vite (当前代码可复用)
- PaddleOCR (本地 OCR)
- @electron/remote (窗口管理)

**实现步骤：**
1. 创建 Electron 项目
2. 集成当前 Web 代码
3. 添加截图功能 (mss/robotjs)
4. 添加全局快捷键
5. 添加游戏内覆盖层

### 方案 C：混合方案 (最佳)

**Web 版 + 本地截图工具**

1. **Web 版** - 保持当前纯前端应用
2. **本地截图工具** - 小型 Electron 应用
   - 监听游戏窗口
   - 自动截图
   - 上传到 Web 版
   - 或本地保存

## 🛠️ Electron 桌面版实现

### 项目结构

```
where-winds-meet-gear-analyzer/
├── web/                    # 当前 Web 版
│   └── src/
├── electron/               # Electron 主进程
│   ├── main.js
│   ├── preload.js
│   └── screenshot.js       # 截图模块
├── package.json            # Electron 配置
└── electron-builder.yml    # 打包配置
```

### 核心代码

**main.js - 主进程**
```javascript
const { app, BrowserWindow, globalShortcut, screen } = require('electron');
const { screenshot } = require('./screenshot');

function createWindow() {
  const win = new BrowserWindow({
    width: 1280,
    height: 800,
    webPreferences: {
      preload: path.join(__dirname, 'preload.js'),
      nodeIntegration: false,
      contextIsolation: true,
    },
  });

  // 加载 Web 版
  win.loadURL('https://zuoshiyue.github.io/where-winds-meet-gear-analyzer/');
}

app.whenReady().then(() => {
  createWindow();

  // 注册全局快捷键 F12 截图
  globalShortcut.register('F12', () => {
    screenshot();
  });
});
```

**screenshot.js - 截图模块**
```javascript
const robot = require('robotjs');
const fs = require('fs');
const path = require('path');

function screenshot() {
  const { width, height } = screen.getPrimaryDisplay().workAreaSize;
  const img = robot.screen.capture(0, 0, width, height);
  
  const savePath = path.join(app.getPath('userData'), 'screenshots');
  const filename = `screenshot_${Date.now()}.png`;
  
  img.save(path.join(savePath, filename));
  
  // 发送到 Web 版
  mainWindow.webContents.send('screenshot-taken', filename);
}
```
