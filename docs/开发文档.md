# 开发文档

## 🚀 快速开始

### 环境要求
- Python 3.10+
- Node.js 18+ (可选，用于前端)
- Git

### 安装步骤

```bash
# 1. 克隆项目
git clone https://github.com/zuoshiyue/where-winds-meet-gear-analyzer.git
cd where-winds-meet-gear-analyzer

# 2. 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 初始化数据库
python main.py --init-db

# 5. 运行应用
python main.py
```

## 🔧 配置说明

### config.yaml 配置示例

```yaml
# OCR 配置
ocr:
  model_dir: ./models/ocr
  lang: ch
  use_gpu: true
  det_db_thresh: 0.3
  det_db_box_score: 0.5

# 截图配置
screenshot:
  hotkey: F12
  output_dir: ./data/screenshots
  format: png
  quality: 95

# 数据库配置
database:
  path: ./data/database/gear.db
  echo: false

# 飞书配置 (可选)
feishu:
  enabled: false
  app_token: ""
  table_id: ""

# 职业配置
classes:
  - name: 剑客
    preferred_sets:
      - 剑心套装
      - 疾风套装

      attack: 1.0
      crit_rate: 0.8
      crit_dmg: 0.9

      - 霸刀套装
      - 铁血套装
    stat_weights:
      attack: 1.0
      strength: 0.7
      crit_rate: 0.6
```

## 🎯 评分系统

### 评分公式
```python
总分 = Σ(属性值 × 属性权重) × 套装加成 × 品质系数

属性权重：根据职业配置
套装加成：2 件套 1.1x, 4 件套 1.25x
品质系数：白 1.0, 绿 1.1, 蓝 1.2, 紫 1.35, 金 1.5
```

### 评分等级
- S: ≥ 90 分 (极品)
- A: 80-89 分 (优秀)
- B: 70-79 分 (良好)
- C: 60-69 分 (一般)
- D: < 60 分 (较差)

## 🐛 常见问题

### OCR 识别不准确
- 检查图像质量
- 调整 OCR 阈值参数
- 添加更多训练样本

### 截图区域偏移
- 重新校准游戏窗口位置
- 检查分辨率设置
- 更新区域配置文件

### 飞书同步失败
- 检查 App Token 配置
- 确认 Bitable 权限
- 查看网络状态
