## 📋 功能概述

### 什么是胚子？
**胚子** = 未经过叠音强化的原始装备，只有**1 个基础词条**

### 什么是叠音？
**叠音** = 装备强化过程，每次消耗"叠音石"增加/提升词条
- **最大叠音次数：** 5 次
- **最终词条数量：** 最多 4 个词条

### 核心功能
1. **胚子预过滤** - 判断是否值得培养
2. **叠音追踪** - 记录每次强化过程
3. **成长评估** - 实时建议继续/止损

## 📊 叠音追踪系统

### 追踪流程

```
1. 胚子评估 → S/A 级 → 开始追踪
2. 每次叠音 → 记录新词条 → 重新评分
3. 对比前后 → 显示提升幅度 → 给出建议
4. 达到毕业 → 停止追踪 → 生成报告
```

### 装备评级标准

根据**有效词条数**和**核心词条数**评定：

```
推荐度分数 = 有效词条数 × 2 + 核心词条数 × 3

- 有效词条：权重 ≥ 0.8
- 核心词条：权重 ≥ 1.2
```

| 分数 | 评级 | 说明 |
|------|------|------|
| ≥12 | 毕业候选 | 可以停止培养 |
| 8-12 | 优质散件 | 继续提升 |
| <8 | 过渡装备 | 考虑止损 |

### 继续培养建议逻辑

**建议继续：**
- 叠音等级 < 3 且 有效词条少
- 核心词条未达到 2 个
- 还有明显提升空间

**建议止损：**
- 已经是毕业候选（≥12 分）
- 核心词条已达标（≥2 个）
- 有效词条为 0 且叠音≥2 次

## 🔧 使用方法

### 1. 进入胚子评估模式

```
分析面板 → 选择"🌱 胚子评估"模式
```

### 2. 上传/识别胚子

```
- 方式 1：OCR 识别装备截图
- 方式 2：手动输入词条信息
```

### 3. 查看评估结果

系统会显示：
- **胚子评级** (S/A/B/C)
- **胚子评分** (当前/期望)
- **权重分析** (流派适配度)
- **培养建议** (是否值得培养)

### 4. 开始叠音追踪（可选）

如果评级为 S 或 A：
```
点击"🎯 开始叠音追踪"
→ 每次叠音后输入新词条
→ 系统自动对比评分变化
→ 实时建议继续/止损
```

### 5. 导出培养报告

```
点击"📊 导出培养报告"
→ 生成完整的培养历程
→ 包含每次叠音的评分变化
→ 最终装备评估
```

## 📈 评分计算公式

### 基础公式

```
装备评分 = Σ(词条数值 × 权重系数)
```

### 词条收益百分比

```
词条收益 (%) = (加入词条后的最终伤害期望 - 原最终伤害期望) ÷ 原最终伤害期望 × 100%
```

### 一级属性转化

```
劲：每 1 点 = 0.17 最小外攻 + 1.41 最大外攻
敏：主要关联闪避、格挡 (生存向)
势：关联部分武学增伤/减伤
```

## 🎯 最佳实践

### 1. 胚子筛选优先级

```
S 级胚子 → 全力培养（资源优先）
A 级胚子 → 正常培养（日常资源）
B 级胚子 → 过渡使用（不投入过多）
C 级胚子 → 直接分解（回收资源）
```

### 2. 叠音止损点

```
- 叠音 2 次后仍无有效词条 → 止损
- 叠音 3 次后只有 1 个核心词条 → 考虑停止
- 叠音 4 次后未达到毕业候选 → 评估是否继续
```

### 3. 资源分配建议

```
叠音石分配：
- 70% 用于 S 级胚子
- 25% 用于 A 级胚子
- 5% 用于 B 级胚子（过渡）
- 0% 用于 C 级胚子
```
