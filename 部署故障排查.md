# GitHub Pages 部署故障排查

## 问题现象

GitHub Actions 持续失败，Setup Node 步骤报错。

## 已尝试的修复

1. ✅ 移除 cache-dependency-path 配置
2. ✅ 生成并提交 package-lock.json
3. ✅ 移除 npm 缓存配置
4. ✅ 简化 workflow 配置

## 可能的问题

### 1. GitHub Pages 未启用

**症状：** Pages 站点不存在 (404)

**解决方法：**
1. 访问仓库：https://github.com/zuoshiyue/where-winds-meet-gear-analyzer/settings/pages
2. 在 **Source** 部分选择 **GitHub Actions**
3. 保存设置

### 2. 需要手动触发首次部署

**解决方法：**
1. 访问：https://github.com/zuoshiyue/where-winds-meet-gear-analyzer/actions/workflows/deploy-web.yml
2. 点击 **"Run workflow"** 按钮
3. 选择 **main** 分支
4. 点击 **"Run workflow"**

### 3. Node.js 版本问题

**尝试：** 修改 `.github/workflows/deploy-web.yml` 中的 node-version：

```yaml
- name: Setup Node
  uses: actions/setup-node@v4
  with:
    node-version: '18'  # 尝试使用 18 而不是 20
```

### 4. npm ci 失败

**尝试：** 改用 npm install

```yaml
- name: Install dependencies
  working-directory: ./web
  run: npm install
```

## 当前配置

```yaml
name: Deploy to GitHub Pages

on:
  push:
    branches:
      - main
    paths:
      - 'web/**'
  workflow_dispatch:  # 支持手动触发

permissions:
  contents: read
  pages: write
  id-token: write

jobs:
  deploy:
    environment:
      name: github-pages
      url: ${{ steps.deployment.outputs.page_url }}
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: Setup Node
        uses: actions/setup-node@v4
        with:
          node-version: '20'

      - name: Install dependencies
        working-directory: ./web
        run: npm ci

      - name: Build
        working-directory: ./web
        run: npm run build

      - name: Setup Pages
        uses: actions/configure-pages@v4

      - name: Upload artifact
        uses: actions/upload-pages-artifact@v3
        with:
          path: ./web/dist

      - name: Deploy to GitHub Pages
        id: deployment
        uses: actions/deploy-pages@v4
```

## 手动部署方案

如果 GitHub Actions 持续失败，可以使用手动部署：

```bash
cd web

# 安装依赖
npm install

# 构建
npm run build

# 安装 gh-pages
npm install -D gh-pages

# 部署
npx gh-pages -d dist
```
