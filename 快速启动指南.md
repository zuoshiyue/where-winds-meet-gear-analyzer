## 📸 使用指南

### 1. 上传截图

1. 点击 **"📸 截图上传"** 标签
2. 拖拽游戏截图到上传区域，或点击选择文件
3. 支持批量上传多张截图
4. 点击 **"开始识别"** 按钮

### 2. OCR 识别

- 系统自动使用 Tesseract.js 进行中文识别
- 识别过程中显示进度条
- 识别完成后自动解析装备数据

### 3. 查看装备

1. 切换到 **"📋 装备列表"** 标签
2. 查看已识别的装备
3. 使用筛选和排序功能
4. 双击查看详情

### 4. 装备分析

1. 切换到 **"📊 装备分析"** 标签
2. 选择职业 (通用/输出/坦克)
3. 查看评分和排序
4. 点击装备查看详细评分和建议

### 5. 数据统计

1. 切换到 **"📈 数据统计"** 标签
2. 查看装备总数、平均等级
3. 查看品质分布和类型分布图表

### 6. 数据管理

1. 切换到 **"⚙️ 设置"** 标签
2. 导出数据 (JSON/CSV)
3. 导入备份数据
4. 清空所有数据

## ⚠️ 注意事项

### OCR 识别准确率

- **受截图质量影响** - 请使用清晰的游戏截图
- **光线影响** - 确保截图亮度适中
- **字体大小** - 游戏内字体不宜过小
- **识别结果可修正** - 可在详情页手动修改

### 浏览器兼容性

| 浏览器 | 最低版本 | 推荐度 |
|--------|---------|--------|
| Chrome | 90+ | ⭐⭐⭐⭐⭐ |
| Edge | 90+ | ⭐⭐⭐⭐⭐ |
| Firefox | 88+ | ⭐⭐⭐⭐ |
| Safari | 14+ | ⭐⭐⭐ |

### 数据备份

- **定期导出** - 建议每次使用后导出 JSON 备份
- **浏览器缓存** - 清除缓存会丢失本地数据
- **跨设备同步** - 通过导入/导出功能迁移数据

## 🔧 故障排查

### GitHub Pages 无法访问

**问题：** 404 错误

**解决方法：**
1. 等待 1-2 分钟，部署需要时间
2. 检查 GitHub Actions 状态
3. 确认部署工作流成功完成

### OCR 识别失败

**问题：** 识别结果为空或错误

**解决方法：**
1. 检查截图清晰度
2. 确保截图包含装备信息
3. 尝试调整截图角度和亮度
4. 手动输入装备数据

### 数据丢失

**问题：** 刷新页面后数据消失

**原因：** IndexedDB 数据被清除

**解决方法：**
1. 从备份导入数据
2. 避免清除浏览器缓存
3. 定期导出备份
